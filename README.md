# Strongbox Vault 🔐

> **Your personal vault for managing passwords, cards, notes, goals, and more.**

Strongbox Vault is a secure, client-side web application designed to be your central hub for managing sensitive personal information. It runs entirely in your browser, with all data stored locally using IndexedDB, putting you in full control of your privacy and security.

## ✨ Key Features

-   **Secure Credential Storage:** Store and manage usernames, passwords, and PINs for all your online accounts.
-   **Notepad:** Create rich-text notes organized within custom folders. Supports checklists, formatting, and more.
-   **Goal Tracking:** Set and track personal goals with categories and a simple "done" toggle.
-   **Strong Security:** All data is encrypted locally using AES-256-GCM. You are in complete control of your encryption key.
-   **Lock Screen:** Protect your vault with a password. The app will auto-lock after a period of inactivity.
-   **Panic Lock:** Quickly lock the app by pressing the `ESC` key twice (when enabled).
-   **QR Code Generator:** Generate QR codes from any text or URL, with options for colors, sizes, and logos.
-   **Search & Filter:** Easily find what you need with live search and category filters.
-   **Backup & Restore:** Export your entire vault as a JSON backup or import it to restore or migrate your data.
-   **CSV Import/Export:** For the Vault section, you can import and export credentials via CSV for easy migration.
-   **Theme Support:** Switch between light and dark themes.
-   **Offline Ready:** Works offline after the first visit and can be installed as a Progressive Web App (PWA).
